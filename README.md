# scripts-against-vuemanity

A Cards Against Humanity clone written with Vue.js.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Runs the WebSocket server
```
npm run server
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
